# Job Application Tracker

## Overview

**Job Application Tracker** is a simple interactive web application designed to help users organize and manage job applications visually. The project presents multiple job posts as cards, allowing users to update the status of each application based on the outcome of their job search process.

Each job card includes action buttons that allow users to categorize the application. By interacting with these buttons, users can move job posts into different sections such as **Interview** or **Rejected**, making it easier to keep track of application progress.

The project was built using HTML5, JavaScript, TailwindCSS, and DaisyUI, with a focus on practicing DOM manipulation and dynamic UI updates.

## Features

* Job application posts displayed as interactive cards
* Buttons on each card to update the application status
* Ability to move job cards into **Interview** or **Rejected** sections
* Dynamic interface powered by JavaScript DOM manipulation
* Structured layout styled with TailwindCSS and DaisyUI
* Clear visual separation of application statuses

## Challenges

One of the main challenges in this project was working with **DOM manipulation using JavaScript** to dynamically move job cards between sections. Managing event handling and ensuring that the correct card updates its position required careful structuring of the JavaScript logic.

Another challenge was maintaining a clear and organized user interface while dynamically updating elements on the page. Ensuring that the card transitions between different sections remain smooth and logically structured required thoughtful implementation of DOM operations and component layout.
